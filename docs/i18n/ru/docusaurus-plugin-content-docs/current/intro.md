# NeoMovies API v2

NeoMovies API v2 — это serverless REST API на Rust, задеплоенный на Vercel.

## Ключевые особенности

- **Аутентификация** — только через [Neo ID SSO](https://id.neome.uk)
- **Медиаданные** — Kinopoisk API и TMDB 
- **База данных** — MongoDB (пользователи, избранное, watch later, синхронизация прогресса, refresh-токены)
- **Деплой** — Vercel Serverless Functions (Rust)
- **CORS** — разрешён для всех источников (`*`)
- **v2** — чистые ответы медиа, фильтрованный поиск

## Базовые URL

```
https://api.neome.uk/api/v1   (legacy, обратная совместимость)
https://api.neome.uk/api/v2   (новые, чистые ответы)
```

## Формат ответов

Все эндпоинты возвращают JSON. Успешные ответы оборачиваются в конверт:

```json
{
  "success": true,
  "data": { ... }
}
```

Ошибки:

```json
{
  "error": "описание ошибки"
}
```

## HTTP-коды

| Код | Значение |
|-----|----------|
| 200 | Успех |
| 400 | Неверный запрос |
| 401 | Не авторизован |
| 404 | Не найдено |
| 500 | Внутренняя ошибка |
| 502 | Ошибка внешнего сервиса |
